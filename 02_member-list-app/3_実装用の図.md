# 実装用の図

## ユースケース図
```puml
@startuml
left to right direction
actor "チーム" as team
actor "社員" as member
rectangle アプリ {
  usecase "チームを管理する" as case1
  usecase "社員をチームに追加する" as case2
  usecase "社員をチームから削除する" as case3
  usecase "社員を管理する" as case4
}
team -- case1
team -- case2
team -- case3
member -- case4
@enduml
```

## クラス図
```puml
@startuml
class チーム {
  - ID : 文字列
  - 名称 : 文字列
  - 紹介文 : 文字列
  + チームを管理する() : void
  + 社員をチームに追加する() : void
  + 社員をチームから削除する() : void
}
class 社員 {
  - ID : 文字列
  - 名前 : 文字列
  - 連絡先 : 文字列
  - オンラインステータス : 真偽
  - 所属チームID : 文字列
  + 社員を管理する() : void
}
チーム "1" -- "0.. *" 社員 : 所属する
@enduml
```
