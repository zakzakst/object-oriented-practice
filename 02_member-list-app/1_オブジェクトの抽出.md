## オブジェクトの多重性を特定する
```dot
digraph {
  graph [
    rankdir = LR,
  ];
  node [
    shape = box,
    style = "filled",
    fillcolor = "#eeeeee",
    margin = 0.2
  ];

  社員 [fillcolor = "#ff8888"];
  チーム [fillcolor = "#ff8888"];

  チーム -> 社員 [dir = forward, arrowhead = box];
  社員 -> 名前 [dir = none];
  社員 -> 連絡先 [dir = none];
  社員 -> オンラインステータス [dir = none];
  チーム -> 紹介文 [dir = none];
  チーム -> 名称 [dir = none];
}
```


## メインオブジェクトに付随するオブジェクトをプロパティとする
```mermaid
classDiagram
  class チーム {
    名称
    紹介文
  }
  class 社員 {
    名前
    連絡先
    オンラインステータス
  }
  チーム --* 社員
```
