## メインオブジェクトに付随するオブジェクトをプロパティとする
```mermaid
classDiagram
  class 会議室 {
    名称
    場所
    広さ
    備品
    利用ステータス
  }
  class 予約 {
    日時
    会議室
  }
  会議室 --* 予約
```
